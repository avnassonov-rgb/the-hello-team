# THE HELLO Team — План производства

Внутреннее веб-приложение: вход по общему паролю, вкладки (Главная — заглушка, План производства — рабочая, Kaspi магазин — заглушка), общее хранилище данных на сервере.

## Что внутри

- `server/` — серверная часть на Node.js (без сторонних библиотек).
- `public/` — то, что видит браузер: страница входа, страница приложения, стили, расчётный модуль.
- `data/state.json` — здесь сервер хранит загруженные файлы и настройки. Один файл — общий для всех, кто заходит по ссылке.
- `package.json` — описание проекта для Node.js.

## Как запустить у себя для проверки

Нужен установленный Node.js (версия 18 или новее).

```
cd ДАШБОРД
node server/server.js
```

Откроется на `http://localhost:3000`. Пароль входа по умолчанию: **thehello2026**.

Пароль и секретный ключ для cookie можно задать через переменные окружения (см. файл `ИНСТРУКЦИЯ_ПУБЛИКАЦИЯ.md` — там написано, как это сделать на хостинге, без программирования).

## Как пользоваться

1. Войти по общему паролю.
2. На вкладке «План производства» в шапке нажать на плитку «не отгруженные» и выбрать xls-файл из 1С, затем плитку «универсальный отчёт» и выбрать второй файл.
3. Файлы читаются прямо в браузере и отправляются на сервер — дальше все, кто откроет эту же ссылку, видят готовый расчёт без повторной загрузки.
4. Кнопки «Печать плана» и «Печать всего» — наверху над таблицами плана.
5. Список «неснижаемый остаток» и настройки ключевых слов — в раскрывающихся блоках над планом, там же кнопки «Сохранить».

Подробно про перенос на свой домен — в файле `ИНСТРУКЦИЯ_ПУБЛИКАЦИЯ.md`.
